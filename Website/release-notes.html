Current release Version 1.0 · June 2026

Ivory Notes 1.0

The first public version is built around fast recording, private on-device transcription, organized notes, and Apple Watch capture.

Platform iPhone, iPad, Apple Watch

Built for the Apple ecosystem

Ivory Notes keeps capture simple on mobile and wrist, then keeps review, storage, settings, and support easy to find.

Version history

What is included.

Release notes will be updated as new builds ship to the App Store.

1.0
Initial release June 2026

Private voice notes with study tools

  • Record lectures, meetings, ideas, and study sessions.
  • Transcribe speech on device without an Ivory Notes account.
  • Generate summaries, study guides, flashcards, and clean note titles.
  • Organize recordings by subject, favorites, source, and newest first.
  • Record from Apple Watch and transfer audio back to iPhone.
  • Review storage details and delete audio while keeping note text.
